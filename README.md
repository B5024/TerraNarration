TerraNarration (泰拉旁白)

让你的泰拉瑞亚之旅不再寂寞！

TerraNarration 为你的角色注入了“灵魂”。它不再是一个只会挥舞工具的工具人，而是一个会在冒险中随时吐槽、感慨和惊慌的鲜活角色。

核心功能：

    动态吐槽系统：在挖矿、战斗、受伤或生命值危急时，角色头顶会弹出极具临场感的浮动旁白。

    沉浸式体验：告别冷冰冰的伤害数字，体验口语化的角色内心戏。

    完全自定义：模组在 Documents/My Games/Terraria/tModLoader/ModConfigs/MyQuotes.json 生成独立的台词库，你可以随时编辑 JSON 文件来创作属于你自己的角色性格！

目前包含的台词场景：

    枯燥挖矿：“这块石头怎么这么硬？”、“挖矿真累啊...”

    遭遇偷袭：“哎哟！谁在暗算我？”、“没看到我在忙吗！”

    濒死挣扎：“眼前有点发黑...”、“药水！快给我药水！”
